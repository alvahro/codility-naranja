Requiere: version lts de npm y NodeJS

La aplicación "se installa" ejecutando el siguiente comando para installar los módulos y dependencias que usa la applicación (Linux o MAc):

$ npm install

La aplicación se corre con el siguiente comando (se incia el servidor web en el puerto 3000 y la app de NodeJS):

$ node server
